## 🔥 Por que o Mixxer é "Tuff"?

* **Interface Brutalista:** Sem ícones fofos. Apenas poder bruto.
* **Engine de Caos:** Simulações físicas que desafiam a lógica da realidade.
* **Renderização de Elite:** Se o seu computador não esquentar, você não está fazendo certo.
* **Workflow Imprevisível:** Às vezes o software decide o que você vai animar.

## 🛠️ Funcionalidades (ou quase isso)

1.  **Modelagem por Grito:** Fale com o seu PC e veja os vértices se moverem.
2.  **Modo Pesadelo:** Ativa shaders que a mente humana ainda não compreende.
3.  **Exportação Instantânea:** Formatos de arquivo que só o Mixxer lê.

## 🚀 Como instalar

1.  Tenha coragem.
2.  Baixe o executável (quando ele decidir existir).
3.  Abra e reze para o seu GPU.
